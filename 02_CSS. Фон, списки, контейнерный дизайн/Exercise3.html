Блок навигации

  • Ширина 15em
  • Должен присутствовать цвет фона
  • Располагаеся слева
  • Блок навигации наполните контентом достаточным, чтобы по высоте он начинал захватывать вторую статью, как показано в данном примере

Статья 1

Размер блока:
Занимает всю досупную ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседей равным .2em, при помощи свойства margin-bottom.

Статья 2

Размер блока:
Занимает всю досупную ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседей равным .2em, при помощи свойства margin-bottom. Картинка вставлена при помощи тега, плавание организовано по правому краю внутри данной статьи для данной картинки.

Нижний блок, подвал. Занимает всю доступную ширину. Высота определяется помещённым внурь контентом.
